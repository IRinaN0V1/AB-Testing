# A/B тестирование: влияние цвета фона сайта на поведение пользователей

## Описание проекта

В данном проекте проведён **A/B тест** для оценки влияния цвета фона сайта на поведение пользователей. Исследование выполнено на синтетическом датасете, имитирующем посещения розничного сайта пользователями из Великобритании (https://www.kaggle.com/datasets/adarsh0806/ab-testing-practice).

**Гипотеза:** изменение цвета фона сайта с белого на чёрный повысит вовлечённость пользователей (время на сайте, просмотры страниц) и конверсию.

**Эксперимент:**
- **Группа A (контрольная):** белый фон сайта (текущая версия)
- **Группа B (тестовая):** чёрный фон сайта (новая версия)

**Целевые метрики:**
- Время на сайте (секунды)
- Количество просмотренных страниц
- Конверсия (целевое действие: покупка)

## Данные

### Характеристики
- **5000 записей**
- **7 колонок:**
  - `User ID` — идентификатор пользователя
  - `Group` — группа (A — контроль, B — тест)
  - `Page Views` — количество просмотренных страниц
  - `Time Spent` — время на сайте (секунды)
  - `Conversion` — совершил ли целевое действие (Yes/No)
  - `Device` — тип устройства (Desktop/Mobile)
  - `Location` — регион Великобритании (England, Scotland, Wales, Northern Ireland)

### Первичный анализ
- Пропуски отсутствуют
- Группы сбалансированы: A — 2519, B — 2481
- Рандомизация корректна: распределение устройств и локаций в группах практически идентично

### 2. Проведена проверка на баланс групп по устройствам и локациям
### 2. Проведен расчёт ключевых метрик

| Метрика | Группа A (белый) | Группа B (чёрный) | Lift |
|---------|------------------|--------------------|------|

## Ключевые выводы

**Конверсия** — единственная метрика, на которую действительно повлиял чёрный фон (p < 0.0001). Рост с 5% до 14% не случаен.
**Время на сайте** и **просмотры страниц** изменились незначимо (p > 0.05) — различия случайны.

Чёрный фон **не делает сайт "интереснее"** (время не выросло), но **значительно увеличивает конверсию**.


## Заключение

Проведённый A/B тест показал, что **изменение цвета фона на чёрный статистически значимо увеличивает конверсию в 2.8 раза**, при этом не влияя на время на сайте и просмотры страниц. Это означает, что дизайн влияет на принятие решения о покупке, а не на вовлечённость в контент.

Результаты могут быть использованы для оптимизации интерфейса и увеличения выручки без дополнительных затрат на трафик.
